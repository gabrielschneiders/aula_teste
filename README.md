\# ⚙️ Primeira Experiência com Configurações



Este repositório registra minha primeira experiência trabalhando com configurações e organização inicial de um projeto. O objetivo foi entender melhor como estruturar um ambiente de desenvolvimento e como arquivos de configuração ajudam a organizar e controlar o comportamento da aplicação.



---



\## 📚 Sobre o Projeto



Durante este projeto, explorei conceitos importantes para a base de qualquer aplicação, focando principalmente na organização e configuração do ambiente.



Alguns pontos trabalhados:



\- Estrutura inicial de um projeto

\- Organização de arquivos de configuração

\- Separação entre código e configurações

\- Primeiras boas práticas de desenvolvimento



---



\## 🎯 Objetivo



O objetivo deste projeto é servir como parte do meu processo de aprendizado, registrando minhas primeiras experiências com configuração e estrutura de projetos.



---



\## 🧠 Aprendizados



Com esta experiência, aprendi:



\- A importância de organizar bem um projeto desde o início

\- Como arquivos de configuração ajudam a tornar aplicações mais flexíveis

\- Como documentar projetos usando um `README.md`
